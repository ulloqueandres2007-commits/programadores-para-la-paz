Durante este curso aprendí cómo usar herramientas tecnológicas para desarrollar soluciones que apoyen la participación ciudadana. Comprendí la importancia de Git y GitHub para organizar proyectos y registrar cambios. También aprendí conceptos básicos de Node.js, Express y el funcionamiento de servidores. Conocí la utilidad de las bases de datos, la autenticación y la automatización mediante herramientas como n8n y Telegram. Además, entendí la importancia de proteger la información y trabajar de forma organizada. Estos conocimientos pueden aplicarse para crear plataformas útiles para las comunidades y mejorar la colaboración entre las personas.

El login permite identificar a un usuario antes de darle acceso a la plataforma. Algunas rutas deben estar protegidas porque contienen información privada o funciones administrativas. La autenticación ayuda a controlar quién puede realizar determinadas acciones. No se deben exponer contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles. Esto protege la privacidad y la seguridad de la comunidad.
